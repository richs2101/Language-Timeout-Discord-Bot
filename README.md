# Custom langugae timeout discord bot

This code is made by [Dave Davidson](https://github.com/richs2101).
This is a custom language timeout discord bot that can do timeouts if anyone speaks Hindi in a English channel. If anyone speaks English in Hindi channel, it'll timeout them automatically.

```bash
1. Add your bot token and channel id which is required in the language_bot.py file.
2. pip install requirements.txt
3. python bot.py
4. Enjoy!
